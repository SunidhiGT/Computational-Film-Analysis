# Computational-Film-Analysis
Overview: This is a project that analyses a film's structure using scene-level emotional metadata and computer vision. It essentially extracts variations in brightnes/intensity, color palette, scene pacing to quantify cinematic rhythm 
===========================
#Key Features:
===========================
- Frame by frame brightness analysis (OpenCV)
- 10-second rolling intensity windows
- Dominant RGB color extract of full video 
- Emotion frequency and duration analysis
- Pacing classification based on average shot length
- Data visualisation with Matplotlib

Technologies used: Python, OpenCV, NumPy, Matplotlib

Why this is important: Film editing and emotional pacingare studied qualitatively. However, this project looks at how computational metrics can analyse rhythm and atmospher
